    功能特性 | Features

    云盘服务的主要功能说明。| Main features of the cloud drive service.

    Upload | 上传

    文件上传 | File Upload

    支持大文件上传，上传前需要输入密码进行身份验证。支持单文件 < 10GB 的上传限制。| Supports large file uploads with password verification. Single file upload limit < 10GB.

    password protected secure
    Download | 下载

    文件下载 | File Download

    下载文件无需密码，任何人都可以下载已上传的文件。支持批量下载和分块下载。| Download files without password, anyone can access uploaded files. Supports batch and resumable downloads.

    public access resumable
    Directory | 目录

    目录管理 | Directory Management

    支持创建和组织目录，可以按项目或类别保存文件。支持目录压缩下载。| Organize files in directories by project or category. Support for compressed directory downloads.

    organization archive

    文件分享 | File Sharing

    可以生成分享链接，与他人共享文件而无需上传权限。支持设置分享链接的过期时间。| Generate share links to share files with others without upload permission. Support for link expiration settings.
