云盘 / Cloud Drive

这个页面直接嵌入远程服务器上的云盘界面。文件会存到 Azure VM 的磁盘里，适合大文件上传、下载和长期保存。上传需要密码，下载不需要。
This page embeds the cloud drive running on the remote server. Files are stored on the Azure VM disk, which is suitable for large uploads, downloads, and long-term storage. Uploading requires a password, while downloads do not.

云盘 / Cloud Drive

上传、下载和文件管理都直接连接到你的远程服务器。这样空间更大，传输也更直接，适合备份、素材和大文件。

大空间 / More space 快速传输 / Fast transfers 远程服务器 / Remote VM
入口已就绪 下面的窗口会直接打开云盘界面。如果嵌入加载失败，可以用单独打开按钮进入。文件都保存在 Azure VM 上，不占用本地空间。上传需要密码，下载不需要密码。 打开云盘 / Open Cloud Drive

使用方式 / How it works

  • 云盘由远程服务器提供存储，适合更大的文件容量。
  • 上传和下载会直接走远程服务器，避免绕回本地。
  • 如果你需要更高安全性，后续可以再加登录口令。

访问提示 / Access tips

  • 建议优先在桌面浏览器使用，拖拽上传更顺手。
  • 上传时请输入你设置的密码；下载不需要密码。
  • 如果你更新了隧道地址，只需要替换这一页里的链接。
  • 这页是入口，真正的文件管理界面在远程服务器上运行。
